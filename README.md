# Tiles Gallery

## 📌 Project Purpose
A modern tile gallery website where users can explore and view different tile designs.

## 🌐 Live URL
https://assignment-8-phi-three.vercel.app/

## 🚀 Features
- Responsive design (mobile, tablet, desktop)
- Dynamic photo gallery
- Google authentication
- User profile system
